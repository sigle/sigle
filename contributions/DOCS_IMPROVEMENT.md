## About

Sigle is a secure, open-source publishing platform built for Web3 creators.  
It helps writers, NFT communities, and crypto-native teams publish content, grow audiences, and manage their online presence with decentralized-friendly workflows.

## Why Sigle

- **Open-source**: transparent codebase and community-driven development.
- **Creator-focused**: designed for Web3 storytelling and community building.
- **Production-ready UX**: modern writing and publishing experience.
- **Extensible**: contributions are welcome across product, design, and engineering.

## Contributing

We welcome all community contributions.

Before opening a pull request, please read [CONTRIBUTING.md](CONTRIBUTING.md) for:

- local setup instructions,
- branching and commit conventions,
- pull request requirements,
- review and release workflow.

If you plan a larger feature or refactor, open an issue first to align on scope.
